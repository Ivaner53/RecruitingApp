## Описание
Данное веб-приложение было разработано в рамках учебной практики. Программа представляет собой платформу для взаимодействия между работодателями и соискателями. Работодатели могут публиковать вакансии, просматривать отклики соискателей, а соискатели – искать подходящие вакансии и отправлять отклики с сопроводительным письмом.

## Установка и запуск
Приложение является веб-сайтом и не требует установки на устройство. Для работы достаточно современного браузера и доступа к сети (или локального сервера).
Варианты запуска:

Скачайте архив с исходным кодом из раздела Releases.

Распакуйте архив в любую папку.

Откройте файл index.html в браузере – приложение запустится в режиме демонстрации с локальным хранилищем (localStorage).

Для полноценной работы с серверной частью и базой данных SQLite необходимо установить .NET 8 SDK, перейти в папку проекта и выполнить dotnet run, после чего открыть браузер по адресу http://localhost:5000.

## Системные требования:

Современный браузер (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari)

Для серверного режима: Windows, Linux или macOS с установленным .NET 8 Runtime.

## Использование
Приложение "Интернет-рекрутинг" предоставляет следующие возможности:

Основные функции:

Регистрация – создание учётной записи с выбором роли (Соискатель или Работодатель). Для работодателя дополнительно указывается компания.

Авторизация – вход в систему по email и паролю.

Просмотр вакансий – список всех вакансий с кратким описанием и датой публикации (доступен даже без авторизации).

Поиск – фильтрация вакансий по названию или названию компании.

Детальный просмотр – при клике на карточку открывается полная информация о вакансии (описание, требования, данные работодателя).

Для соискателя:

Отклик на вакансию – отправка сопроводительного письма.

Просмотр своих откликов – список всех отправленных откликов с датой и текстом письма.

Для работодателя:

Создание вакансии – добавление новой вакансии с указанием названия, описания и требований.

Просмотр своих вакансий – список созданных вакансий с возможностью просмотра количества откликов.

Просмотр откликов на свои вакансии – для каждой вакансии отображаются ФИО соискателя, текст его сопроводительного письма и дата отклика.

## Работа с приложением:

При запуске отображается главная страница со списком вакансий.

В верхней панели находятся кнопки входа/регистрации (для гостей) или приветствие, кнопка "Мои вакансии/отклики", "Обновить" и "Выйти" (для авторизованных пользователей).

Все действия (создание вакансии, отклик) выполняются через модальные окна.

Данные сохраняются на сервере в базе данных SQLite (при серверном запуске) или в localStorage браузера (при локальном запуске).

## Разработка
Данный проект разрабатывается в рамках учебной практики и предназначен для образовательных целей. Проект создаётся одним разработчиком и не предусматривает привлечение сторонних участников.

## Технические детали:

Язык программирования (серверная часть): C# (ASP.NET Core 8)

Фронтенд: HTML5, CSS3, JavaScript

База данных: SQLite (с использованием Entity Framework Core)

Среда разработки: Visual Studio 2022 / VS Code

## Архитектура проекта:

Клиент-серверная архитектура: фронтенд обращается к REST API, реализованному на ASP.NET Core.

Аутентификация на основе сессий (или JWT – опционально).

Ролевая модель: Гость, Соискатель, Работодатель.

Данные хранятся в реляционной базе данных SQLite.
